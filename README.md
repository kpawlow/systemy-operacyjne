# Konfiguracja usług Linux — Technik Informatyk

Repozytorium zawiera przykłady konfiguracji najpopularniejszych usług systemu Linux wykorzystywanych podczas egzaminów zawodowych na kierunku Technik Informatyk.

## 📚 Zakres materiału

W repozytorium będą pojawiać się przykłady konfiguracji między innymi:

- Samba
- SSH
- Apache2
- Nginx
- MariaDB / MySQL
- DHCP
- DNS
- FTP
- Docker
- Firewall (UFW / iptables)
- Cron
- RAID
- NFS
- LDAP
- Active Directory Samba
- VPN
- Home Assistant
- Konfiguracja sieci Linux
- Uprawnienia i użytkownicy
- Skrypty Bash

---

## 🎯 Cel repozytorium

Celem repozytorium jest:

- przygotowanie do egzaminu zawodowego,
- nauka administracji systemami Linux,
- szybki dostęp do gotowych konfiguracji,
- pokazanie konfiguracji krok po kroku,
- zebranie najczęściej pojawiających się usług egzaminacyjnych w jednym miejscu.

---

## 🐧 Systemy

Przykłady będą wykonywane głównie na:

- Ubuntu Server
- Debian
- Rocky Linux / AlmaLinux

---

## ⚙️ Forma przykładów

Każdy przykład będzie zawierał:

- opis działania usługi,
- instalację pakietów,
- konfigurację krok po kroku,
- przykładowe pliki konfiguracyjne,
- restart usług,
- test działania,
- typowe błędy i ich rozwiązania.

---

## 📖 Dla kogo?

Repozytorium jest przeznaczone dla:

- uczniów kierunku Technik Informatyk,
- osób przygotowujących się do egzaminów INF.02 / INF.03 / INF.04,
- początkujących administratorów Linux,
- osób uczących się usług sieciowych.
